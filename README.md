# ProCaPPIS

Prostate Cancer Protein-Protein Interaction Prediction System.

## Description

ProCaPPIS is a machine learning-based tool for predicting protein-protein interactions related to prostate cancer.

## Installation

Clone the repository and install the required dependencies:

```bash
git clone https://github.com/tissueandcells/ProCaPPIS.git
cd ProCaPPIS
pip install -r requirements.txt
```

## Usage

Run the application:

```bash
python app.py
```
